# Автоматизация тестирования API (JSONPlaceholder) на Python

Пет-проект QA-инженера по автоматизации REST API с использованием современного стека на Python.

## Стек технологий
* **Язык:** Python 3
* **Тест-раннер:** PyTest
* **Запросы:** Requests
* **Тестовые данные:** Faker (динамическая генерация данных)
* **CI/CD:** GitHub Actions (автозапуск тестов при каждом push)
