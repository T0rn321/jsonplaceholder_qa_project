# Автоматизация тестирования API (JSONPlaceholder) на Python

Пет-project по автоматизации REST API с использованием современного стека на Python.

## Стек технологий
* **Язык:** Python 3
* **Тест:** PyTest
* **Запросы:** Requests
* **Тестовые данные:** Faker (динамическая генерация данных)
* **CI/CD:** GitHub Actions (автозапуск тестов при каждом push)

Цели проекта:
Создание тестового фреймворка

Покрытие ключевой бизнес-логики (CRUD)

Внедрение процессов CI/CD
